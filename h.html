Help

How to install game?

1- Download Minetest from here

2- Download the last version of Urminaca from download page

3- Extract Urminaca and copy Urminaca Folder.

4- Go to minetest -> games and paste Urminaca folder there.

installing is done :)

How to play?

1- Open minetest from minetest -> bin and run minetest

2- make a new world from start game tab

3- write your world's name and select Urminaca in game section and click on Create

4- from left you can see Creative and Enable Damage. if you want to play survival check

Enable damage and make sure two more buttons are clean

5- Select your world, click on play and enjoy your game :)
